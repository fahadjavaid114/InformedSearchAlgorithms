# InformedSearchAlgorithms

## How to Run
1. Install Python 3.x
2. Install Pygame: `pip install pygame`
3. Run: `python pathfinder.py`

## How to Run
1. Download or clone this repository.
2. Open a terminal in the folder containing the code.
3. Run the application using:

python pathfinder.py

## Controls
1. Click on the right panel to switch algorithms (GBFS or A*) and heuristics (Manhattan or Euclidean).
2. Click on empty grid cells to toggle walls (obstacles) on and off.
3. Click Run to watch the algorithm find the path dynamically.
4. Click Reset to clear the current search and start over.
